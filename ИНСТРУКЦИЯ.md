# Инструкция: размещение Claude Chat на GitHub Pages

## Что ты получишь
- Личный URL вида `https://твой-логин.github.io/claude-chat/`
- Приложение работает как PWA — устанавливается на телефон
- Хранилище IndexedDB — практически неограниченное (гигабайты)
- HTTPS — обязательное условие для PWA и API
- Бесплатно, навсегда

---

## Шаг 1. Создай аккаунт на GitHub

1. Открой https://github.com на телефоне или компьютере
2. Нажми **Sign up**
3. Придумай логин (например `aleksei-peshkov` или любой другой)
4. Подтверди email

---

## Шаг 2. Создай репозиторий

1. После входа нажми **+** (верхний правый угол) → **New repository**
2. **Repository name:** `claude-chat`
3. Выбери **Public** (обязательно для бесплатного GitHub Pages)
4. Поставь галочку **Add a README file**
5. Нажми **Create repository**

---

## Шаг 3. Загрузи файлы

В репозитории нажми **Add file** → **Upload files**

Загрузи все эти файлы (они лежат в папке `claudechat`):
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

Нажми **Commit changes** (зелёная кнопка внизу).

---

## Шаг 4. Включи GitHub Pages

1. В репозитории нажми **Settings** (шестерёнка вверху)
2. Слева найди раздел **Pages**
3. В разделе **Source** выбери:
   - Branch: **main**
   - Folder: **/ (root)**
4. Нажми **Save**

GitHub покажет: *"Your site is live at https://твой-логин.github.io/claude-chat/"*

⏱ Обычно занимает 1–3 минуты.

---

## Шаг 5. Открой на телефоне

1. Открой Chrome на Android
2. Перейди по адресу `https://твой-логин.github.io/claude-chat/`
3. Chrome покажет баннер **"Установить как приложение"** — нажми
4. Приложение появится на рабочем столе как обычное приложение

Если баннер не появился — нажми ⋮ (меню Chrome) → **Добавить на главный экран**

---

## Шаг 6. Введи API ключ

1. В приложении нажми кнопку **⚿ key**
2. Вставь ключ из https://console.anthropic.com
3. Нажми **Сохранить**

Ключ хранится в localStorage — вводить нужно только один раз (или при переустановке).

---

## Обновление приложения

Если захочешь обновить код:
1. Зайди на github.com в свой репозиторий
2. Нажми на файл `index.html` → карандаш ✏ (Edit)
3. Замени содержимое
4. Нажми **Commit changes**
5. Через 1–2 минуты сайт обновится

На телефоне приложение обновится автоматически при следующем запуске.

---

## Смена телефона / Backup

1. В приложении: боковая панель → **↓ Экспорт JSON**
2. Сохрани файл в Google Drive или Telegram Saved Messages
3. На новом телефоне: открой приложение → **↑ Импорт JSON**
4. Все диалоги восстановлены

---

## Хранилище IndexedDB vs localStorage

| | localStorage | IndexedDB (это приложение) |
|---|---|---|
| Лимит | ~5 MB | Зависит от свободного места на телефоне |
| Типичный лимит | 5 MB | 1–50+ GB |
| Потеря данных | При сбросе браузера | Сохраняется дольше |

---

## Приватность

- Код открытый (Public репозиторий), но твои диалоги и API ключ нигде не хранятся на GitHub
- Диалоги: только у тебя в IndexedDB на телефоне
- API ключ: только в localStorage на телефоне
- Трафик: напрямую с твоего телефона на api.anthropic.com

---

## Выбор модели

В хедере приложения есть переключатель **Sonnet / Opus**:

| Модель | Скорость | Стоимость | Когда использовать |
|---|---|---|---|
| **Sonnet** | быстрее | дешевле (~3× ) | Большинство задач |
| **Opus** | медленнее | дороже | Сложный анализ, рассуждения |

Имена без даты (`claude-sonnet-4-5`) — автоматически обновляются когда выходит новая версия.
Если хочешь зафиксировать версию — замени на `claude-sonnet-4-5-20251001`.
